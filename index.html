Projet MapBox

HTML | CSS | JS (Jquery - Ajax) | API (Mapbox - datagouv.fr)

Ce petit projet met en application l'utilisation de l'API Mapbox (déployement d'une carte avec ses fonctionnalités), combinée avec l'API datagouv.fr (récupération des données des entreprises).
Le but est de saisir le nom d'une organisation dans la barre de recherche, et la map va se centrer sur l'oganisation en affichant des informations sur celle-ci.
Pour se faire, j'ai du utiliser les technologies mentionnées ci-dessus.

Ce mini projet utilise la v1 de l'API datagouv.fr.
Tout manquement, ou fausse information proviennent des données renvoyées par cette API.
Un temps de réponse court nécéssite une connexion correcte.
